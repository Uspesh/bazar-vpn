## Параметры окружения(.env)

- BOT_KEY - токен для тестового бота.
- DB_NAME - имя вашей базы данных
- REDIS_HOST - обычно ставится localhost. Советуем так и оставить
- REDIS_PORT - порт редиса, советуем оставить как по умолчанию 6379
- BAZAR_VPN_BOT_KEY - токен вашего основного бота. Будет использоваться если DEBUG = False
- YOOMONEY_TOKEN - токен для подключения юмани. Ссылка на то как его получить - https://lozerix.com/threads/4545/
- SENTRY - api токен сервиса отслеживания ошибок Sentry
- AMOUNT - стоимость подписки
- TG_CHANNEL - канал, подписка на который будет отслеживаться ботом
- DEBUG - режим работы, может быть True/False. Если True то будет использован BOT_KEY для тестов и добавления новый фитч.
